File an overseas entity update statement.

Use this service to check and update information about an overseas entity and its beneficial owners or managing officers.

All overseas entities must file an overseas entity update statement at least once each year to confirm that the information on the Companies House register is up to date.

Verification checks

If there have been any changes to any of the overseas entity's beneficial owners or managing officers, these changes must be verified by a UK-regulated agent. This must be done no more than 3 months before this update statement is filed. The UK-regulated agent must have an agent assurance code issued by Companies House.

Before you start

You'll need:

  • to sign in to or create a Companies House account
  • the Overseas Entity ID
  • the overseas entity's authentication code
  • the agent assurance code, if you're the UK-regulated agent that carried out the verification checks
  • a credit or debit card to pay the £XX fee
