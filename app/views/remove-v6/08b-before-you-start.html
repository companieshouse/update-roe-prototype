Before you start

Once an overseas entity is removed, its Overseas Entity ID will no longer be valid. It will no longer be able to buy, sell, transfer, lease or charge property or land in the UK, unless it re-registers.

Authentication code

You will need the overseas entity's authentication code to file this application. Find out how to get the authentication code if you don't already have it.

Check if any information needs to be updated

All information about the overseas entity, its beneficial owners and managing officers must be correct on the date of this application.

If you need to update any information, you can do that as part of this service. Any changes must have been verified by a UK-regulated agent no more than 3 months before this application is submitted.
