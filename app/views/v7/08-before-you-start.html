Before you start

Authentication code

You will need the overseas entity's authentication code to file this update statement. If you need to request one, search for the overseas entity on the Companies House register, then select 'Request an authentication code'. We'll send the code to the email address that we have on record for the overseas entity.

Agent assurance codes for verification checks

If there have been any changes to any of the overseas entity's beneficial owners or managing officers, these changes must be verified by a UK-regulated agent. This must be done no more than 3 months before this update statement is submitted.

The UK-regulated agent will need to provide their agent assurance code to confirm they have authorisation to carry out verification checks.
