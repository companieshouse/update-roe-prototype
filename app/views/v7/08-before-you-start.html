Before you start

Authentication code

You will need the overseas entity's authentication code in order to file an update statement using this service.

If you do not have it, read the guidance on how to request an authentication code.

Verification checks

If any of the overseas entity's information has changed during this update period, these changes must be verified by a UK-regulated agent. This must be done no more than 3 months before the date of this update statement.

The UK-regulated agent must have an agent assurance code, which confirms they have authorisation to carry out verification checks.

If you are not changing any information, you do not need to complete any verification checks for this service.
