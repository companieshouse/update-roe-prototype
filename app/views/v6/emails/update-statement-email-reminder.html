Overseas entity ASHCROFT LIMITED
Overseas Entity ID OE001485

It's nearly time to file your update statement. The date of your first update statement is one year from the date that the overseas entity was registered. You have 14 days from the statement date to file.

You must file your update statement by 24 August 2023.

You need to file an overseas entity update statement every year to confirm that the information we hold is correct and up to date. You must file an update statement even if nothing has changed.

It’s a criminal offence if you do not file your update statement.

1. Request an authentication code
The authentication code is a unique 6 character code, and we'll ask for it every time someone files online on behalf of the overseas entity to make sure they’re authorised to do so.

To get an authentication code, search for the overseas entity on the Find and update company information service and select ‘Request an authentication code’ on the entity’s overview page. We’ll send the code to the email address we have on record for the overseas entity.

www.find-and-update.company-information.service.gov.uk

2. Complete your verification checks
A UK-regulated agent must complete verification checks on all beneficial owners and managing officers of the overseas entity, if your information has changed since you registered. This must be done no more than 3 months before you file your update statement.

Verification checks only need to be completed on information that's being updated since your first registration. If nothing has changed and you’re not updating any information, you will not need to complete any verification checks.

3. File an overseas entity update statement
After you've had your authentication code and completed your verification checks, you’ll be ready to file your overseas entity update statement.

Go to GOV.UK for more information and to file your update statement:
www.gov.uk/guidance/overseas-entities

If you do not file your update statement on time:

  • You’ll be committing a criminal offence and you could get a fine
  • Your registration on the Register of Overseas Entities will no longer be valid and you will not be able to buy, sell, transfer, lease or charge your property or land in the UK
  • We’ll add a note to your public record that says you have not filed your update statement and that your registration is no longer valid
