Before you start

Once an overseas entity has been removed, its Overseas Entity ID will no longer be valid. This means it will no longer be able to buy, sell, transfer, lease or charge property or land in the UK.

Authentication code

You will need the overseas entity's authentication code to file this application. Find out how to get the authentication code if you don't already have it.

Verification checks

As part of this application, you need to tell us if there have been any changes to the information we hold about the overseas entity, its beneficial owners or managing officers. These changes must have been verified by a UK-regulated agent no more than 3 months before this application is submitted.

If you are not changing any information, you will not need to complete any verification checks.
