Get ready to file an overseas entity update statement

Overseas entity ASHCROFT LIMITED
Overseas Entity ID OE000063

Every year you need to file an overseas entity update statement to confirm that the information we hold is up to date.

There are some things you can do now to start preparing to file your update.

Request an authentication code
The authentication code is a 6 character code that every overseas entity needs to request. We ask for it every time someone files online for the overseas entity â€” it confirms to us that they are authorised to do so. If you do not have an authentication code yet, you can request one:
www.gov.uk/request-authentication-code/LDJFHLJDFHLSH

Make sure verification checks are completed
A UK-regulated agent must complete verification checks on all beneficial owners and managing officers of the overseas entity. This must be done no more than 3 months before this statement is filed.

If you're the UK-regulated agent carrying out verification checks, you'll need an agent assurance code. If you do not have a code yet, you can request one:
https://www.gov.uk/guidance/agent-assurance-codes

File an overseas entity update statement
After you've received your authentication code and the verification checks have been completed, you'll be ready to file your overseas entity update statement:
www.gov.uk/file-overseas-entity-update

For more information, read the guidance on overseas entities: www.gov.uk/guidance/overseas-entities
